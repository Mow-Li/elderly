# 相亲相爱桌面 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
相亲相爱桌面

### 1.2 产品定位
专为老年人设计的鸿蒙系统桌面启动器，简化操作、防止误触、强化亲情联系。

### 1.3 目标用户
- 主要用户：65 岁以上老年人
- 配置者：子女/家人（帮助设置和配置）

### 1.4 核心价值
- **防误触**：固定桌面，避免老人误操作进入系统
- **易操作**：大图标、家人照片、一键呼叫
- **强联系**：快速联系家人，降低使用门槛

## 2. 功能需求

### 2.1 桌面锁定功能 🔒

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 禁止上划 | 禁用系统手势，防止进入任务栏 | P0 |
| 禁止 Home 键 | 禁用或拦截 Home 键（需系统权限） | P0 |
| 禁止返回键 | 在桌面界面禁用返回键 | P0 |
| 退出方式 | 仅能通过 APP 内设置页面退出 | P0 |
| 退出验证 | 无需密码，直接退出 | P0 |

### 2.2 联系人图标功能 📞

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 照片图标 | 使用家人照片作为桌面图标 | P0 |
| 图标布局 | 3列网格布局，最多6个联系人 | P0 |
| 点击弹窗 | 点击图标弹出操作选择菜单 | P0 |
| 电话呼叫 | 直接拨打预设电话号码 | P0 |
| 微信电话 | 跳转微信发起语音通话 | P0 |
| 微信视频 | 跳转微信发起视频通话 | P0 |

### 2.3 APP 快捷方式功能 📱

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 自定义 APP | 可选择显示哪些 APP（如抖音） | P0 |
| APP 图标 | 显示选定 APP 的图标和名称 | P0 |
| 一键启动 | 点击直接打开对应 APP | P0 |
| 数量限制 | 最多显示6个 APP 快捷方式 | P0 |

### 2.4 设置页面功能 ⚙️

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 进入方式 | 点击设置按钮进入 | P0 |
| 联系人管理 | 添加/编辑/删除联系人及照片 | P0 |
| APP 管理 | 选择显示哪些 APP | P0 |
| 退出桌面 | 退出锁定模式，返回系统桌面 | P0 |
| 语音设置 | 开启/关闭语音播报 | P0 |
| 字体大小 | 调整全局字体大小 | P0 |

### 2.5 语音播报功能 🔊

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 点击播报 | 点击联系人图标时语音播报姓名 | P0 |
| 开关控制 | 可在设置中开启/关闭语音 | P0 |
| 语速调节 | 支持调节语音速度 | P1 |

### 2.6 字体大小调整功能 📏

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 多档位选择 | 提供 5 档字体大小（小/中/大/超大/特大） | P0 |
| 默认设置 | 默认为大号字体 | P0 |
| 全局生效 | 调整后所有界面文字同步变化 | P0 |

## 3. 技术需求

### 3.1 平台要求
- **系统**：HarmonyOS 4.0+
- **开发语言**：ArkTS
- **开发工具**：DevEco Studio

### 3.2 系统权限
| 权限 | 用途 | 说明 |
|------|------|------|
| `ohos.permission.CALL_PHONE` | 拨打电话 | 必需 |
| `ohos.permission.GET_BUNDLE_INFO` | 获取 APP 信息 | 必需 |
| `ohos.permission.START_ABILITY` | 启动其他 APP | 必需 |
| `ohos.permission.BIND_LAUNCHER` | 作为启动器 | 必需 |
| `ohos.permission.MICROPHONE` | 语音播报（TTS） | 必需 |

### 3.3 关键技术点
1. **Launcher 模式**：已配置 entity.system.home
2. **窗口覆盖**：使用全屏窗口
3. **微信跳转**：通过 Want 跳转微信
4. **数据存储**：使用 Preferences 存储配置

## 5. 数据模型

### 5.1 联系人 Contact
```typescript
{
  id: string;           // 唯一ID
  name: string;         // 姓名
  photoUri: string;     // 照片路径
  phone: string;        // 电话号码
  wechatId: string;    // 微信ID（可选）
  position: number;    // 桌面位置 0-5
}
```

### 5.2 应用快捷方式 AppShortcut
```typescript
{
  bundleName: string;   // 应用包名
  abilityName: string;  // Ability名
  appName: string;      // 显示名称
  iconUri: string;     // 图标路径
  enabled: boolean;     // 是否显示
  position: number;    // 桌面位置 0-5
}
```

### 5.3 设置 Settings
```typescript
{
  fontSize: FontSize;     // 字体大小枚举
  voiceEnabled: boolean;  // 语音开关
  voiceSpeed: number;      // 语音速度 0.5-2.0
}
```
